## 🧮 Dashboard ADM – Malotes, Crachás e Brindes

**Ferramentas:** Excel, Power BI (layout), tabelas, gráficos, VBA (automatizações).

**Descrição:**  
Dashboard em tema dark para acompanhar malotes recebidos/enviados, estoque atual de brindes e quantidade de entradas, com filtros por ano e mês.

**Destaques técnicos:**
- Cards de KPI para malotes recebidos, enviados, estoque de brindes e entradas de brindes.
- Gráfico de linha mostrando a evolução mensal de malotes.
- Gráfico de rosca com distribuição de crachás por tipo (colaborador, visitante, prestador).
- Gráfico de barras para movimentação de brindes.
